5.2 O gráfico é acompanhado de uma descrição longa

Sobre o requisito

Gráficos resultantes de análise de dados deverão ser acompanhados da tabela de dados que lhe deu origem, de forma a preservar o acesso à informação completa.

Método de avaliação

  1. Encontrar elementos gráficos no domínio do site. Podemos considerar neste critério elementos como gráficos, organogramas, fotografias/imagens, infografias, fluxogramas, etc.
  2. No caso dos gráficos, verificar se estão acompanhados de uma tabela de dados OU
  3. A tabela está visível na própria página, colocada numa página à parte ou num ficheiro externo (excel, PDF) e o conteúdo dessa tabela seja acessível pelo leitor de ecrã.
  4. No caso dos outros elementos gráficos, devem ter uma descrição longa.

Referências de boas práticas

  • W3C Tutorial - Imagens complexas
  • Boas práticas de gráficos com tabelas no Relatório de Monitorização do Período 2020/21

Técnica WCAG aplicada no website exemplo do workshop

  • G73: Fornecer uma descrição longa em outro local com um link para ela imediatamente adjacente ao conteúdo não textual
  • G74: Fornecer uma descrição longa em texto perto do conteúdo não textual, com uma referência à localização da descrição longa na descrição curta
  • G92: Fornecer uma descrição longa para conteúdo não textual que tenha a mesma finalidade e apresente as mesmas informações
