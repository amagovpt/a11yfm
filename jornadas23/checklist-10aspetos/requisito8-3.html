8.3 Quando se retira a CSS, deve ser possível reconhecer a semântica dos diversos elementos

Sobre o requisito

Os elementos que estruturam o conteúdo devem estar semanticamente bem estruturados, usando os elementos de HTML apropriados a cada tipo de conteúdo, como títulos, parágrafos, listas, etc... Sem estilo é possível analisar a diversidade semântica do documento. Outro aspeto que é possível analisar é a semântica dos elementos. Isto é, se os conteúdos são parágrafos, títulos, listas.

Método de avaliação

  1. Com o plugin Web Developer selecionar a opção CSS > Desativar todos os estilos (disabled all styles)
  2. Verifique se é possível reconhecer a semântica dos diversos elementos da página, por exemplo ao desativar o CSS, um botão deve aparentar ser um botão, um link deve aparentar ser um link (sublinhado ou a cor azul), etc...

Técnica WCAG aplicada no website exemplo do workshop

  • Técnica G115: Usando elementos semânticos para marcar estrutura
  • Técnica H49: Usando marcação semântica para marcar texto enfatizado ou especial
