10.1 Nos ficheiros PDF é possível, no mínimo, extrair o conteúdo textual para formato TXT

Sobre o requisito

Os ficheiros PDF devem ter o seu texto inteiramente extraível para que se possa passar o respetivo conteúdo para um processador de texto sem perda de informação.

Método de avaliação

  1. Abrir um ficheiro PDF do website e tentar selecionar o texto para ver se é possível copiar ou não desse ficheiro para um processador de texto (exemplo: Word).
  2. Após copiar o texto, garantir que não houve perda de informação. Verifique se todo o texto foi convertido e está na ordem de leitura correta.
