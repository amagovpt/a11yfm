8.1 Quando se retira a CSS, todos os elementos HTML devem alinhar à esquerda

Sobre o requisito

Quando se desativam todos os estilos visuais, o conteúdo da página é apresentado alinhado à esquerda e apresenta-se de forma linear. Ao executar o procedimento proposto (retirar a CSS) ficamos só com a estrutura. Como todos os elementos HTML alinham à esquerda, se, porventura, existirem ainda resíduos de estilo aplicados no HTML (p.e. text-align=“center”) o conteúdo não irá alinhar, todo ele, à esquerda. Este procedimento permite verificar se há resíduos de estilo na estrutura.

Método de avaliação

  1. Com o plugin Web Developer selecionar a opção CSS > Desativar todos os estilos (disabled all styles)
  2. Quando o conteúdo não encosta à esquerda, podem existir estilos misturados nos próprios atributos, mesmo na própria linha de código de HTML, resultando em conflito e apresentar elementos com CSS aplicado.
  3. O requisito apenas chumba apenas se o leitor de ecrã não ler devidamente o conteúdo.

Técnica WCAG aplicada no website exemplo do workshop

  • G140: Separar informações e estrutura da apresentação para permitir diferentes apresentações
