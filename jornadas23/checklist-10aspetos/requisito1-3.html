Requisito 1.3 As imagens-link, caso existam no menu, devem ter o correspondente equivalente alternativo em texto

Sobre o requisito

As imagens corretamente legendadas permitem ser interpretadas como texto, tornando todas as opções de navegação acessíveis.neste requisito avaliamos imagens-link que fazem parte da lista de opções disponível no menu. O logotipo é avaliado no critério 5.3 imagens-link

Método de avaliação

  1. Para confirmar se o código HTML tem o atributo alt preenchido corretamente, é necessário posicionar o rato nas imagens, clicar no botão do lado direito do rato e selecionar a opção “Inspecionar” OU
  3. Com a ferramenta Web Developer, selecionar Imagem > Mostrar atributo alt (display alt atributes).
  4. As imagens podem ou não ser decorativas:
    - Se o conteúdo da imagem possui informações importantes para o entender o conteúdo da página, o seu texto alternativo deverá estar preenchido apropriadamente;
    - Se o conteúdo da imagem contém apenas informação visual para fins estéticos ou de efeito, o seu texto alternativo deve ser nulo (alt="").
  5. Verificar se as imagens do website possuem o atributo alt e se estão preenchidos corretamente.

Técnica WCAG aplicada no website exemplo do workshop

  • Técnica G94: Fornecer alternativas de texto curto para conteúdo não textual que atenda ao mesmo propósito e apresente as mesmas informações que o conteúdo não textual
  • Técnica H37: Usando altatributos em imgelementos
