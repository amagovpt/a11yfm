7.2 O vídeo ou o áudio deve conter preferencialmente legendas fechadas sincronizadas. Caso não seja possível, no mínimo, deve disponibilizar-se uma transcrição textual

Sobre o requisito

O uso de legendas fechadas destina-se essencialmente a pessoas surdas. Recomendam-se para a produção das referidas legendas técnicas de tradaptação conhecidas para o efeito bem como o enriquecimento das legendas de sons cuja mensagem não seja percetível visualmente (por exemplo, o toque de uma campaínha de uma porta). Para vídeos com mensagens eminentemente visuais (por exemplo, um vídeo com música de fundo que passa um conjunto de mensagens apenas percetíveis à visão), os mesmos devem ter uma versão equivalente alternativa com produção de audiodescrição. A audiodescrição é fundamental para que pessoas cegas ou com baixa visão possam percecionar a mensagem veiculada.

Método de avaliação

  1. Verificar quantos vídeos existem no site e quantos tipos de reprodutores de multimédia são utilizados. Links que direcionam para vídeos externos em outras plataformas, como o YouTube, são considerados para análise.
  2. Verificar se o player de vídeo contém Legendas fechadas (closed captions): se o vídeo tiver legendas no player, com a opção de ativar e desativar legendas (legendas fechadas), é a melhor prática porque permite: controlar o tamanho e a a cor das legendas, ter uma transcrição correta, utilizar o leitor de ecrã para ler essas legendas/transcrição correta e utilizar a pesquisa para encontrar os termos da transcrição.
  3. Verificar se o player de vídeo contém Legendas abertas: se o vídeo tiver legendas/texto embutido no vídeo (legendas abertas), é importante garantir que o áudio acompanhe o conteúdo. Pode não ser acessível para pessoas que precisam de alterar a cor ou aumentar o tamanho da legendas. (ex: pessoas com problemas de visão e de audição).
  4. Verificar se o player de vídeo contém Legendas automáticas: se o website tiver legendas automáticas e que está fiel ao conteúdo apresentado, pode ser aproveitada a transcrição dessas mesmas legendas para construir legendas fechadas para o player.
  5. Verificar se o player de vídeo contém Não tem legendas: deve-se confirmar se existe uma transcrição textual do vídeo.
