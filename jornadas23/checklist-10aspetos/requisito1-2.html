Requisito 1.2 É possível selecionar as opções e as subopções do menu quer com rato quer com teclado

Sobre o requisito

Deve ser possível percorrer a estrutura de navegação quer com um dispositivo apontador, quer com o teclado.

Método de avaliação

  1. Utilizar as teclas TAB e SHIFT+TAB para passar pelos vários elementos e perceber se o menu e submenus são selecionados sequencialmente ou não. (Menu de 1º nível deve passar para as opções de 2º nível).
  2. Utilizar a tecla ENTER para ver as subopções do menu e a tecla ESC para fechar a subopção que está aberta.

Observação: Pode ser possível estilizar os comandos de teclado, como por exemplo, utilizar as setas direcionais para navegar pelo conteúdo. Não reprova o requisito, no entando, o desenvolvimento deverá seguir boas práticas recomendadas da W3C. Também é necessário garantir o funcionamento do mesmo com os leitores de ecrã.
